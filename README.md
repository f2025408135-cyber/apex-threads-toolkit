# APEX Threads Toolkit

Professional security research toolkit for a bug bounty researcher targeting the Meta Threads API.

## Installation

You can install this toolkit directly via `pip` so that it is available globally on your system.

```bash
cd apex-threads-toolkit
pip install -e .
```

Once installed, you can use the `apex-harness` and `webhook-strike` commands directly from anywhere!

## Tools
* **[APEX-HARNESS](./apex_harness/README.md):** Automated security test execution engine.
* **[WEBHOOK-STRIKE](./webhook_strike/README.md):** Webhook signature validation bypass testing tool.
